The local path

Source

Each tool begins with a named source, supported input, and review date. The public proof page links back to the contract rather than recreating it.

Draft

Tools can shape a browser-local working artifact. It does not turn a draft into a filing position, authority decision, or stored client record.

Export

A clear label helps a practitioner choose what leaves the browser. A saved trail, assignment, or recurring workflow belongs elsewhere.

Visible before any handoff
